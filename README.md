# Planti

## 요약
- 제2회 경상북도 공공데이터 공모전에서 진행한 프로젝트 입니다.
- **과일별 최적지역 추천**을 통해 경상북도 과일 농가에 **자신의 과수를 심는/분양받는 C2C 중개 서비스 "Planti"** 입니다.
